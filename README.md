# nlp-productivity-app

A productivity web application that allows users to manage tasks using both a graphical interface and a natural language chatbot.

The system is designed to explore how Natural Language Processing (NLP) can improve usability and provide a more human-centered way to organize daily activities.

Add tasks using natural language (e.g., "add buy milk")
View all pending tasks
Chat-based interaction with the system
Simple and clean web interface
SQLite database for persistent storage

Python
Flask (Backend)
SQLite (Database)
JavaScript (Fetch API) (Frontend communication)
HTML/CSS (User Interface)
spaCy (planned for NLP processing)
